# Calculadora de Habilidade Básica - Hero Zero

Projeto de extensão desenvolvido para a Universidade Veiga de Almeida (UVA). A aplicação tem como objetivo validar a lógica matemática de bônus acumulativos e calcular os atributos básicos para a comunidade do jogo Hero Zero.

## O Projeto
O sistema foi desenvolvido em Java, focando em portabilidade e precisão lógica. Durante a fase de testes práticos, as fórmulas implementadas bateram com os valores reais do jogo, apresentando variações mínimas devido à conversão de valores para números inteiros feita pelo próprio ecossistema do game.

## Portabilidade e Ambientes de Teste
O software foi homologado e testado com sucesso em múltiplos ambientes:
* **Linux:** Executado via IDE NetBeans.
* **Windows:** Executado via Prompt de Comando (CMD).

## Como Baixar e Executar (Windows)
Para facilitar a experiência do usuário, o projeto conta com uma estratégia de implantação autônoma (*private runtime*):

1. Acesse a aba **[Releases](https://github.com/AndreLauterjung/projeto-extensao-hero-zero/releases/tag/v1.0.0)** e baixe o arquivo `.zip`.
2. Extraia o conteúdo no seu computador.
3. Dê dois cliques no arquivo executável em lote `.bat`.

**Nota:** O pacote inclui uma pasta JRE de 32 bits embutida, o que significa que o programa roda direto e não exige que você tenha o Java instalado no seu computador (compatível com Windows 32 e 64-bits).
